# HRFlow - HR Management System

HRFlow is a simple Human Resource Management System built using **HTML,
CSS, and JavaScript**.

## Features

-   Employee Management
-   Payroll Management
-   Performance Tracking
-   Attendance Tracking
-   Leave Management
-   Reports Dashboard
-   Settings Page
-   Login System
-   Responsive User Interface

## Project Structure

``` text
HRflow/
│
├── index.html              # Landing page
├── login (1).html          # Login page
├── dashboard.html          # Dashboard
├── employees.html          # Employee management
├── Attendance.html         # Attendance tracking
├── Leave.html              # Leave management
├── Payroll.html            # Payroll management
├── Performance.html        # Performance tracking
├── reports.html            # Reports page
├── settings.html           # Application settings
│
├── style.css               # Main stylesheet
├── script.js               # Main JavaScript file
├── employees.js            # Employee functionality
├── reports.js              # Reports functionality
├── settings.js             # Settings functionality
├── data.js                 # Data handling
└── dummy-data.js           # Sample employee data
```

## Technologies Used

-   HTML5
-   CSS3
-   JavaScript (Vanilla JS)

## Getting Started

1.  Download or clone the repository.
2.  Open `index.html` in your browser.
3.  Log in to access the dashboard and features.
